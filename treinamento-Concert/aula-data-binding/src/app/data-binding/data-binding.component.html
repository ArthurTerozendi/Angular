Interpolação

Utilizando a Interpolação é possivel:

  • Imprimir o valor de uma variável -> {{url}}
  • Somar números -> {{ 1 + 1 }}
  • Chamar alguma função/método -> {{ getTexto()}}
  • Realizar operações booleanas -> {{ true || false && (false && true) }}
